Order Information

What is an Order?

An order represents a purchase request to a supplier. Each order must be associated with a supplier ID and have a specific amount.

Order Status

Pending: Order has been placed but not fulfilled. Completed: Order has been fulfilled and delivered. Cancelled: Order has been cancelled.
